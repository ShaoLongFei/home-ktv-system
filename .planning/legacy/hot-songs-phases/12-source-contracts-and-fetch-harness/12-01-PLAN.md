    Required schemas:
    - `SourceTypeSchema = z.enum(["ktv_first", "support"])`
    - `SourceKindSchema = z.enum(["public_chart", "manual_snapshot"])`
    - `ProviderSchema = z.enum(["qq_music", "cavca", "kugou", "netease", "manual"])`
    - `AdapterSchema = z.enum(["qq_toplist", "kugou_rank_html", "netease_toplist_html", "manual_json"])`
    - `SourceStatusValueSchema = z.enum(["succeeded", "failed", "stale", "skipped"])`

    `SourceDefinitionSchema` must include `id`, `name`, `provider`, `sourceType`, `sourceKind`, `adapter`, `weight`, `enabled`, `required`, `url`, `file`, `expectedMinRows`, `staleAfterDays`, `usableWhenStale`, and optional `headers`.

    Contract rules:
    - `id` matches `/^[a-z0-9][a-z0-9-]*$/`
    - `weight` integer min `1` max `200`
    - `enabled` default `true`
    - `required` default `false`
    - `expectedMinRows` default `1`
    - `staleAfterDays` default `14`
    - `usableWhenStale` default `false`
    - reject header keys `cookie`, `authorization`, `x-token`, and `x-auth-token`
    - reject `public_chart` sources without `url`
    - reject `manual_snapshot` sources without `file`

    Add `SourceManifestSchema` with literal `schemaVersion: "hot-songs.source-manifest.v1"`.
    Add `ManualSnapshotSchema` with literal `schemaVersion: "hot-songs.manual-source.v1"`.
    Add `SourceRowSchema` and `SourceStatusSchema`.
    Export inferred types `SourceDefinition`, `SourceManifest`, `ManualSnapshot`, `SourceRow`, `SourceStatus`, and `SourceStatusValue`.

    - rejection of `headers.authorization`
